### 现有装饰元素

- **背景**：165° 线性渐变（暖奶白 → 杏黄 → 金黄）
- **散点星星**：12 个 `✦` 符号，随机分布在页面四周，金色 + 紫色，6–20px
- **底部山丘**：3 层 SVG 路径，purle 系色，营造景深感

---

## 现有问题

1. **视觉层次单薄**：背景渐变 + 少量星星，中间大块留白，缺少焦点
2. **主标题区域无视觉锚点**：文字周围没有装饰图形支撑
3. **底部山丘孤立**：山丘和主内容之间缺少过渡元素
4. **整体偏静态**：没有动感或层次错落感

---

## 设计方向建议

### 风格参考
参考 `/2026.jpg`（已有概念图）：暖色水彩插画风，泰式文化元素（天灯、莲花、寺庙、大象），金紫配色，手绘质感

### 需要的新元素（任选）

**背景层**
- 柔和的水彩纹理底图（可以是 PNG/WebP，全幅，带透明度）
- 或更丰富的渐变/噪声纹理

**主内容区两侧**
- 左侧：1–2 个泰式天灯（Yee Peng Lantern）插画，金黄色调
- 右侧：莲花、或泰式建筑剪影
- 也可以是更抽象的几何/装饰图形

**标题装饰**
- 标题上方 / 下方的装饰线、光晕、或手写风格符号

**底部**
- 现有山丘可以替换为更丰富的景观插画（有层次感的山、水、植物）
- 或保留山丘，在前景加植物/花朵元素

### 资产格式要求

| 资产 | 格式 | 建议尺寸 | 说明 |
|------|------|---------|------|
| 背景底图 | WebP / PNG | 1440×900px | 需要带透明度或作为底层 |
| 左侧装饰 | SVG / WebP | 自由 | 需要带透明背景 |
| 右侧装饰 | SVG / WebP | 自由 | 需要带透明背景 |
| 底部景观 | SVG / WebP | 1440×300px | 替换现有山丘 SVG |
| 移动端适配 | 同上 | 768px 宽版本 | 可选，如果桌面版太宽 |

### 颜色沿用建议
主色调不变（金紫 + 暖米），新插画元素用同色系即可融入现有渐变背景。

---

## 技术约束

- 框架：Vue 3 + Vite + Tailwind CSS
- 图片放在 `src/assets/` 目录，直接 import 使用
- 背景图用 `background-image: url()` 或 `<img>` absolute 定位
- SVG 可直接内联或 import 作为组件
- 页面目前是**单屏全高**（`min-h-dvh`），暂无滚动 section
